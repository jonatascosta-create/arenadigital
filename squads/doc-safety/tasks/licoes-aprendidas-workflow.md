# Licoes Aprendidas — Workflow

> **Agente:** @especialista-cultura-seguranca | **Elicit:** true | **Output:** Secao de conclusao do documentario

## Pre-requisitos
- Analise SST do @perito-sst (OBRIGATORIO)
- Analise de cultura do @especialista-cultura-seguranca (OBRIGATORIO)
- Contexto historico do @historiador (OBRIGATORIO)
- Analise tecnica do @engenheiro-naval (DESEJAVEL)
- Estrutura narrativa do @roteirista (DESEJAVEL — para integrar no ATO III)

## Objetivo

Gerar a secao de CONCLUSAO do documentario: licoes que o caso ensina sobre Cultura de Seguranca, conectando passado a presente de forma que o publico leve para casa.

## Etapas

### Etapa 1: Sintetizar Inputs
**Acao:** Ler e sintetizar todos os inputs dos outros agentes
- Do @perito-sst: falhas sistemicas identificadas (Dekker + Reason)
- Do @especialista-cultura: classificacao cultural + insights
- Do @historiador: forcas historicas que criaram as condicoes
- Do @engenheiro-naval: decisoes tecnicas que materializaram falhas
**Output:** Mapa de convergencia: onde todos os inputs apontam para as mesmas licoes

### Etapa 2: Definir Licoes (6-8 licoes)
**Acao:** Para cada licao, usar o template:

```
LICAO [N]: "[Titulo provocativo]"
MECANISMO: [Principio de cultura de seguranca violado — Hopkins/Schein/Westrum/Reason]
NO CASO: [Momento concreto do caso com fontes]
HOJE: [Paralelo contemporaneo verificavel]
PERGUNTA: [Provocacao que o publico leva para casa]
VISUAL: [Sugestao de como mostrar no documentario]
```

**Elicit:** "Quantas licoes para a conclusao? (a) 5-6 essenciais, (b) 8-10 abrangentes, (c) 3 impactantes (menos e mais)"

### Etapa 3: Ordenar por Impacto Narrativo
**Acao:** Ordenar as licoes para crescer em impacto
- Comece com licoes mais "esperadas" (compliance, treinamento)
- Termine com licoes mais PROVOCATIVAS (classe, poder, design intencional)
- A ULTIMA licao deve ser a que o publico leva para casa

### Etapa 4: Escrever "E Voce?" (Call to Action)
**Acao:** Escrever o parágrafo final que conecta o caso ao espectador
- Nao e moralismo — e ESPELHO
- Nao e pessimismo — e CONVITE a reflexao
- Tom: provocativo mas respeitoso, urgente mas nao alarmista
**Template:** "O Titanic afundou em 1912. Mas as forcas que o afundaram — [lista] — existem AGORA em [onde]. A pergunta nao e 'como deixaram isso acontecer?' A pergunta e: [pergunta final]."

### Etapa 5: Consolidar Secao de Conclusao
**Output final:** Secao completa pronta para integrar no ATO III do roteiro
- Licoes ordenadas por impacto
- Cada licao com mecanismo + exemplo + paralelo + pergunta
- "E Voce?" como fechamento
- Sugestoes visuais para cada licao

## Quality Gate (doc-chief valida)
- [ ] Minimo 5 licoes com template completo (mecanismo + caso + hoje + pergunta)
- [ ] Cada licao rastreavel a input de outro agente (nao inventada)
- [ ] Licoes ordenadas por impacto narrativo crescente
- [ ] "E Voce?" escrito com tom adequado (provocativo, nao moralista)
- [ ] Sugestoes visuais para ao menos 3 licoes
- [ ] Ponte para presente com paralelos CONCRETOS (nao genericos)
