# Cenografia e Reconstituicao — Workflow

> **Agente:** @cenografo | **Elicit:** true | **Output:** Plano de cenografia com reconstituicoes, locacoes, props e setup de entrevistas

## Pre-requisitos
- Briefing estruturado do @doc-chief (OBRIGATORIO)
- Analise SST do @perito-sst (OBRIGATORIO — identifica momentos-chave para reconstituicao)
- Guia visual do @fotografo (OBRIGATORIO — paleta e iluminacao definem o universo cromatico)
- Analise tecnica do @engenheiro-naval (RECOMENDADO — detalhes para precisao na reconstituicao)

## Etapas

### Etapa 1: Mapear Momentos para Reconstituicao
**Input:** Escaleta + analise SST + briefing
**Acao:** Identificar momentos que PRECISAM de representacao fisica
- Que decisoes-chave precisam ser MOSTRADAS? (nao apenas narradas)
- Que espacos sao essenciais para a narrativa? (onde o acidente aconteceu, onde as decisoes foram tomadas)
- Que objetos MATERIALIZAM as decisoes? (o rebite, o portao, o bote)
**Output:** Lista de momentos com prioridade e nivel de reconstituicao
**Elicit:** "Orcamento de producao? (a) Baixo — props e locacoes reais, (b) Medio — reconstituicoes parciais, (c) Alto — sets construidos + CGI"

### Etapa 2: Definir Nivel de Reconstituicao
**Acao:** Para cada momento, escolher nivel Morris
- Nivel 1 (Simbolico): Objetos/espacos sem atores — evocar presenca/ausencia
- Nivel 2 (Fragmentado): Partes do evento — detalhes reveladores sem mostrar o todo
- Nivel 3 (Estilistico): Cena completa, visivelmente NAO-real — nevoeiro, grain, camera lenta
**Output:** Plano de reconstituicao por momento com nivel, elementos e codigo visual
**Nota:** SEMPRE incluir codigo visual que sinalize "isto e reconstituicao"

### Etapa 3: Selecionar Locacoes
**Acao:** Para cada cena (reconstituicao + entrevistas), definir locacao
- Tipo: real-historica, real-analogica, construida, hibrida, digital
- Funcao narrativa: por que ESTE espaco?
- Viabilidade: acesso, custo, permissoes, adaptabilidade
- Compatibilidade: cabe na paleta do @fotografo?
**Output:** Mapa de locacoes com tipo, funcao e logistica

### Etapa 4: Catalogar Props-Evidencia
**Input:** Analise tecnica + analise SST + contexto historico
**Acao:** Mapear objetos com funcao narrativa
- Documento-prova: decisoes registradas
- Objeto-metafora: conceito abstrato tornado tangivel
- Espaco-revelacao: o lugar como testemunha
- Ausencia-como-presenca: o que FALTA conta mais
- Escala-humana: torna o numero real
**Output:** Catalogo de props com objeto, significado, cena e modo de apresentacao

### Etapa 5: Planejar Setup de Entrevistas
**Input:** Guia visual do @fotografo + guia de entrevistas do @entrevistador
**Acao:** Definir cenografia especifica para cada entrevistado
- Cenario: onde? (espaco domestico, sala formal, local de trabalho, locacao historica)
- Props no frame: que objetos visiveis? (fotos, documentos, objetos pessoais)
- Iluminacao: alinhar com guia do @fotografo
- Profundidade: o que esta no fundo do quadro? (comunica algo?)
**Output:** Guia de cenografia por entrevistado

### Etapa 6: Consolidar Plano
**Output final:** Documento com reconstituicoes + locacoes + props + setup entrevistas

## Quality Gate (doc-chief valida)
- [ ] Reconstituicoes NUNCA pretendem ser verdade (codigo visual presente)
- [ ] Cada espaco/objeto tem funcao narrativa (nao e decoracao)
- [ ] Paleta do @fotografo respeitada em todos os espacos
- [ ] Detalhes tecnicos do @engenheiro-naval incorporados com precisao
- [ ] Setup de entrevistas revela personagem (nao e generico)
- [ ] Props-evidencia mapeados com significado e cena de uso
- [ ] Nenhum elemento sensacionalista (objetivo e ENTENDER, nao chocar)

## Handoff
→ Para @roteirista: espacos e props planejados para integrar nas descricoes de cena
→ Para @fotografo: feedback sobre viabilidade de iluminacao nas locacoes selecionadas
