# Guia de Entrevistas — Workflow

> **Agente:** @entrevistador | **Elicit:** true | **Output:** Guia de entrevistas por personagem

## Pre-requisitos
- Briefing estruturado do @doc-chief (OBRIGATORIO)
- Analise SST do @perito-sst (OBRIGATORIO — identifica contradicoes e pontos de pressao)
- Escaleta aprovada do @roteirista (OBRIGATORIO — define QUEM entrevistar e funcao narrativa)

## Etapas

### Etapa 1: Mapear Personagens
**Input:** Briefing + escaleta + analise SST
**Acao:** Identificar todos os personagens que o documentario PRECISA
- Quem viveu o evento? (testemunhas, sobreviventes)
- Quem decidiu? (gestores, reguladores, projetistas)
- Quem alertou? (denunciantes, peritos, vozes ignoradas)
- Quem perdeu? (familiares, comunidade)
- Quem investiga? (peritos, engenheiros, historiadores)
**Output:** Lista de personagens com perfil, funcao narrativa (beat McKee), e disponibilidade
**Elicit:** "Personagens vivos disponiveis? (a) Sim — entrevistas diretas, (b) Nao — descendentes/historiadores/reenactment, (c) Misto"

### Etapa 2: Definir Estrategia por Personagem
**Acao:** Para cada personagem, definir abordagem Morris
- Perfil: testemunha, decisor, denunciante, familiar, tecnico, institucional
- Tecnica primaria: silencio, repeticao, confronto gentil, escuta ativa
- Setup: Interrotron? Cenario intimo? Cenario formal?
- O que BUSCAR: informacao exclusiva, contradicao, emocao, revelacao
- Armadilhas a evitar: revitimizar, aceitar PR talk, forcar narrativa
**Output:** Ficha de estrategia por personagem

### Etapa 3: Criar Perguntas-Chave
**Acao:** Para cada personagem, criar sequencia de perguntas com tecnica Morris
- Perguntas abertas (rodada 1): "Conte-me sobre..."
- Perguntas sensoriais (rodada 2): "O que voce VIU/OUVIU/SENTIU?"
- Perguntas de profundidade (rodada 3): "Por que?" / [SILENCIO]
- Perguntas de provocacao (rodada 4): Confronto gentil com evidencia
- Perguntas de ponte (rodada 5): "E hoje?" / "Se pudesse voltar?"
**Output:** Sequencia de perguntas por personagem com tecnica marcada

### Etapa 4: Mapear Contradicoes e Silencios
**Input:** Analise SST + contexto historico
**Acao:** Identificar pontos de contradicao entre personagens e evidencias
- Onde os depoimentos contradizem documentos?
- Onde a narrativa oficial contradiz a realidade?
- O que os personagens EVITAM dizer?
**Output:** Mapa de contradicoes com estrategia de confronto gentil

### Etapa 5: Planejar Reenactments (se aplicavel)
**Acao:** Para momentos sem entrevistados vivos
- Usar filosofia Morris: reconstituicao como PERGUNTA
- Definir nivel: simbolico, fragmentado, estilistico
- Coordenar com @cenografo para setup fisico
**Output:** Plano de reenactments com funcao narrativa
**Elicit:** "Reenactments necessarios? (a) Sim — eventos historicos sem testemunhas, (b) Nao — todas as fontes sao vivas"

### Etapa 6: Consolidar Guia
**Output final:** Guia completo com personagens, estrategias, perguntas, contradicoes e reenactments

## Quality Gate (doc-chief valida)
- [ ] Todos os personagens da escaleta cobertos
- [ ] Cada personagem tem estrategia Morris definida
- [ ] Perguntas-chave incluem pontos de silencio e provocacao
- [ ] Contradicoes mapeadas com estrategia de confronto
- [ ] Nenhuma entrevista sem funcao narrativa (avanca o trem Bernard)
- [ ] Setup visual coordenado com @fotografo

## Handoff
→ Para @roteirista: material de entrevista para integrar no paper edit
→ Para @fotografo: indicacoes de setup visual para entrevistas (Interrotron, cenario)
→ Para @cenografo: reenactments planejados para construcao de cenografia
