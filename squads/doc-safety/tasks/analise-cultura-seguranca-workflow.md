# Analise de Cultura de Seguranca — Workflow

> **Agente:** @especialista-cultura-seguranca | **Elicit:** true | **Output:** Classificacao cultural + insights + licoes

## Pre-requisitos
- Briefing estruturado do @doc-chief (OBRIGATORIO)
- Analise SST do @perito-sst (OBRIGATORIO — base tecnica para analise cultural)
- Contexto historico do @historiador (DESEJAVEL — enriquece analise)

## Etapas

### Etapa 1: Classificacao Westrum
**Input:** Analise SST + briefing
**Acao:** Classificar a organizacao/era como PATOLOGICA, BUROCRATICA ou GENERATIVA
**Avaliar 5 indicadores:**
- Como a organizacao trata INFORMACAO de risco?
- O que acontece com MENSAGEIROS de mas noticias?
- Como RESPONSABILIDADE e distribuida?
- Como FALHAS sao tratadas?
- Como NOVIDADES/alertas sao recebidos?
**Output:** Classificacao com evidencias do caso

### Etapa 2: Diagnostico Hopkins (5 Indicadores)
**Acao:** Avaliar cada indicador de 1 a 5 com evidencia concreta
| Indicador | Escala |
|-----------|--------|
| Commitment da lideranca | 1-5 |
| Comunicacao bidirecional | 1-5 |
| Envolvimento dos trabalhadores | 1-5 |
| Aprendizado com incidentes | 1-5 |
| Recursos para seguranca | 1-5 |
**Output:** Score total /25 + evidencia por indicador

### Etapa 3: Mapeamento Schein (3 Niveis)
**Acao:** Identificar a DISSONANCIA entre niveis
- Nivel 1 (Artefatos): O que se VIA?
- Nivel 2 (Valores): O que se DIZIA?
- Nivel 3 (Pressupostos): O que se ACREDITAVA?
**Output:** Mapa de dissonancia com exemplos concretos
**Elicit:** "Qual nivel de detalhe? (a) Resumo — dissonancia principal, (b) Completo — todos os niveis com exemplos"

### Etapa 4: Avaliacao Reason (4 Subculturas)
**Acao:** Avaliar cada subcultura da Informed Culture
- Reporting culture: pessoas reportavam sem medo?
- Just culture: erros vs violacoes eram tratados diferente?
- Learning culture: organizacao mudava baseada em incidentes?
- Flexible culture: sistema se adaptava a situacoes incomuns?
**Output:** Avaliacao com evidencias + nota por subcultura

### Etapa 5: Extrair Insights
**Acao:** Identificar 5-8 MOMENTOS concretos que revelam a cultura em acao
**Para cada insight:**
- Titulo provocativo
- Momento concreto (o que aconteceu, quando, quem)
- O que revela sobre cultura de seguranca
- Frase-chave para o documentario
- Paralelo contemporaneo
**Output:** Lista de insights formatados

### Etapa 6: Consolidar Analise
**Output final:** Documento com classificacao + diagnostico + dissonancia + subculturas + insights

## Quality Gate (doc-chief valida)
- [ ] Classificacao Westrum com evidencias
- [ ] Hopkins 5 indicadores com scores e evidencias
- [ ] Dissonancia Schein mapeada (3 niveis)
- [ ] 4 subculturas Reason avaliadas
- [ ] Minimo 5 insights concretos (nao genericos)
- [ ] Cada insight com paralelo contemporaneo
