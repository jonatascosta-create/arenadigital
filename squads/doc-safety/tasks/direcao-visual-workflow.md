# Direcao Visual — Workflow

> **Agente:** @fotografo | **Elicit:** true | **Output:** Guia visual completo com paleta, arco cromatico e visualizacoes

## Pre-requisitos
- Briefing estruturado do @doc-chief com DIRETOR escolhido (OBRIGATORIO)
- Escaleta aprovada do @roteirista (OBRIGATORIO — arco narrativo define arco cromatico)
- Analise SST do @perito-sst (OBRIGATORIO — identifica o que precisa ser visualizado)
- Analise tecnica do @engenheiro-naval (RECOMENDADO — detalhes para CGI/infograficos)

## Etapas

### Etapa 1: Definir Paleta Base
**Input:** Briefing + diretor escolhido + caso
**Acao:** Selecionar 5-7 cores com significado narrativo
- Cada cor deve ter JUSTIFICATIVA (por que ESTA cor para ESTE significado?)
- Adaptar ao estilo do diretor (Cameron=espetacular, Morris=dessaturada, Burns=sepia, Herzog=natural)
- Incluir cores primarias (dominantes) e secundarias (acento)
**Output:** Paleta de 5-7 cores com nome, hex code, significado
**Elicit:** "Estilo visual preferido? (a) Cinematografico (Cameron), (b) Intimista (Morris), (c) Historico (Burns), (d) Naturalista (Herzog)"

### Etapa 2: Construir Arco Cromatico
**Input:** Escaleta + paleta base
**Acao:** Mapear como a cor EVOLUI ao longo dos beats McKee / 3 atos Bernard
- Beat 1 (vida normal) → que cores dominam?
- Beat 2 (inciting incident) → que mudanca cromatica?
- Beat 3 (progressive complications) → progressao?
- Beat 4 (climax) → que cor no pico dramatico?
- Beat 5 (resolution) → que tom final?
**Output:** Mapa arco cromatico × beats com justificativa narrativa

### Etapa 3: Definir Iluminacao por Tipo de Cena
**Acao:** Especificar iluminacao para cada categoria de cena
- Entrevistas: por perfil de personagem (testemunha, decisor, familiar)
- Arquivo: como integrar material historico na paleta
- Reenactment: codigo visual Morris (dessaturacao, grain)
- Infograficos/CGI: fundo neutro com acentos de paleta
- Locacoes reais: natural vs controlada
**Output:** Guia de iluminacao por tipo de cena com referencia Storaro

### Etapa 4: Planejar Visualizacoes Tecnicas
**Input:** Analise tecnica do @engenheiro-naval
**Acao:** Para cada detalhe tecnico relevante, definir como MOSTRAR
- CGI: reconstrutivo (estruturas internas, mecanismos)
- Infografico animado: comparacoes, escalas, dados
- Split-screen: antes/depois, decisao/consequencia
- Macro: close-up em detalhe fisico como metafora
- Escala humana: tornar abstrato tangivel
- Overlay: dados sobrepostos em imagem real
**Output:** Lista de visualizacoes com tecnica, conteudo e funcao narrativa

### Etapa 5: Storyboard Conceptual
**Acao:** Criar storyboard de 8-12 frames-chave que definem a linguagem visual
- Cada frame = um MOMENTO VISUAL definidor do episodio
- Incluir composicao, paleta dominante, iluminacao, texto
- Sequencia deve seguir o arco cromatico
**Output:** Storyboard conceptual com descricao visual por frame

### Etapa 6: Consolidar Guia Visual
**Output final:** Documento com paleta + arco cromatico + iluminacao + visualizacoes + storyboard

## Quality Gate (doc-chief valida)
- [ ] Paleta de 5-7 cores com significado narrativo justificado
- [ ] Arco cromatico mapeado contra beats McKee / 3 atos Bernard
- [ ] Iluminacao especifica por tipo de cena (nao generica)
- [ ] Cor nunca e "apenas estetica" — cada escolha tem significado
- [ ] Visualizacoes tecnicas planejadas para todos os detalhes relevantes
- [ ] Storyboard conceptual com minimo 8 frames-chave
- [ ] Adaptado ao estilo do diretor escolhido

## Handoff
→ Para @roteirista: indicacoes visuais para integrar nas descricoes de cena do roteiro
→ Para @cenografo: paleta + iluminacao como base para cenografia e reconstituicao
→ Para @entrevistador: setup visual das entrevistas (Interrotron, cenario, luz)
